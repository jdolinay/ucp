UTB Control Package (UCP)
Projekt knihovny pro ridici aplikace s mikropocitaci.  
Aktualne testovano pro mikropocitace rady Freescale HCS08 a pro Kinetis (KL25Z)


Struktura souboru
------------------

doc 		- dokumentace knihovny
examples	- ukazkove programy knihovny
source		- zdrojove kody knihovny (napr. regulatory) vcetne hlavickovych souboru
support		- podpurny kod, napr. uart driver apod.
template 	- "sablony" zdrojovych kodu, ktere uzivatel prida do sve aplikace

Podrobnejsi informace viz doc\ucp.doc
