Detailed Description

MCU-specific code provided by the user.

Note
Toto jsou funkce, ktere vyuziva UCP aplikace. Tyto funkce musi implementovat sam uzivatel knihovny pro svuj konkretni hardware! napr. pokud meri teplotu teplomerem s analogovym vystupem, bude ve funkci ucphal_read_input() cist hodnotu z AD prevodniku.
