Detailed Description

Softwarove PWM.

Note
Obsahuje jednoduchou softwarovou PWM pro realizaci akcniho zasahu pomoci diskretniho pinu mikropocitace. Uzivatel musi zajitit:
  • pravidelne volani funkce pwm_tick
  • implementovat funkci ucphal_write_pin v UCP HAL Podporuje 2 kanaly. Mozno snadno rozsirit na vice kanalu.
